I am learning how to use git and push to github.

I did this by doing the following

  • In your GitHub profile make a new repository.
  • In the command line on your computer cd into your working folder and type : [git init] This will initialize a git repository in your project
  • Next type: [git status] This will list out all the files in your working directory
  • Then type: [git add .] or [git add name-of-file] This adds in all of your files and changes so far into what is called a staging area. You must add these files into a staging area before you can commit them.
  • Then type: [git commit -m 'first commit'], this commits all of your files or the one you specifically named with a message, 'first commit'
  • Next type: [git remote add origin -your github repo URL-]. Your GitHub Repository page will list the repository URL, you can copy/paste it from there into the terminal. This creates a remote, or connection, named 'origin' pointing at the GitHub repository you just made.
  • Then type: [git push -u origin master]. This sends your commits in your 'master' branch to GitHub.

Now you have pushed to GitHub!

To continue making changes and pushing them to GitHub use the following three commands in the terminal

  • [git add .] or [git add name-of file]
  • [git commit -m 'your message']
  • [git commit origin master]
